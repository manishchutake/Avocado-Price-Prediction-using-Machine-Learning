🥑 Avocado Price Prediction using Machine Learning

This project predicts avocado prices using different Machine Learning regression models. 
It includes complete data analysis, model training, and performance comparison.

Project Workflow:
1. Data Exploration (EDA)
2. Data Cleaning & Preprocessing
3. Model Training
4. Model Comparison
5. Best Model Selection

Files in Project:
- data.csv → Dataset used for prediction
- eda.ipynb → Exploratory Data Analysis
- price_regression.ipynb → Regression model training
- model_comparison.ipynb → Comparison of different ML models

Technologies Used:
Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-Learn
Jupyter Notebook

Goal:
To identify the best regression model for accurate avocado price prediction.
